# Pools (Ressourcen-Quotas)

Pools sind Portal-interne Ressourcen-Container, die festlegen, wie viele VMs, CPUs, RAM und Disk eine Gruppe von Nutzern deployen darf.

## Pool erstellen
Gehe zu **System Settings → Nutzer & Rechte → Pools** und klicke auf **+ Pool erstellen**.

## Quoten
Setze Maximalwerte für: VMs, vCPUs, RAM (GB) und Disk (GB). Leer lassen = unbegrenzt.

> **Erfordert** Plus-Lizenz. Core-Edition: 0 Pools.
