# Template-Builder (Packer)

Der Template-Builder erstellt wiederverwendbare Proxmox-VM-Templates mit HashiCorp Packer.

## Build starten
1. Wähle eine Packer-Build-Definition aus der Liste
2. Fülle die Parameter aus (OS-Version, Festplattengröße, Pakete usw.)
3. Klicke auf **Build starten**

## Fortschritt überwachen
Öffne den Job auf der Seite „Logs & Events", um den Packer-Output live zu verfolgen.

## Ergebnis
Ein fertiges Template erscheint im Proxmox-Datenspeicher und wird im Deployment-Dropdown verfügbar.
