# Dashboard

Das Dashboard zeigt eine Echtzeit-Übersicht deines Proxmox-Clusters.

## Inhalt
- **Node-Karten** – CPU-, RAM- und Storage-Auslastung pro Compute-Node
- **VM-Tabelle** – alle VMs und Container mit Status, IP und Ressourcennutzung
- **Ankündigungen** – vom Admin angeheftete Meldungen

## Tipps
- Klicke auf eine Node-Karte, um die VM-Tabelle auf diesen Node zu filtern
- Klicke auf einen VM-Namen, um die Detailseite zu öffnen
- Die Cluster-Statusleiste oben zeigt eine Zusammenfassung des gesamten Clusters
