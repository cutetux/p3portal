# Node Alerting (Plus)

Konfiguriere Schwellwert-Alarme für einen bestimmten Proxmox-Node.

## Alarmregeln
Jede Regel überwacht eine Metrik (CPU, RAM, Disk) und schlägt an, wenn der Wert eine Grenze für eine bestimmte Dauer überschreitet.

## Benachrichtigungskanäle
- Portal-Banner (immer)
- E-Mail (wenn SMTP konfiguriert)
- Webhook (HTTP POST an externe URL)

> **Erfordert** Plus-Lizenz.
