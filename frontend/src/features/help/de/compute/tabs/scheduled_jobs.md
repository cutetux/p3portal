# Zeitgesteuerte Jobs auf Node (Plus)

Konfiguriere zeitbasierte Aktionen für einen bestimmten Proxmox-Node.

## Job-Typen
- **Ansible-Playbook** – Playbook nach Zeitplan ausführen
- **SSH-Befehl** – Shell-Befehl auf einer VM/LXC via SSH
- **Power-Aktion** – VM/LXC starten, stoppen, neu starten

## Zeitplan
Visueller Cron-Picker oder direkter Cron-Ausdruck. Zeiten in der lokalen Zeitzone des Servers.

> **Erfordert** Plus-Lizenz.
