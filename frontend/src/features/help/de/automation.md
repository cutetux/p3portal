# Automation / Zeitgesteuerte Jobs

Die Automation-Seite gibt einen globalen Überblick aller zeitgesteuerten Jobs.

## Filtern
Nach Status (aktiv, gesperrt, deaktiviert) oder Name filtern.

## Job-Typen
- **Ansible-Playbook** – Playbook nach Zeitplan
- **SSH-Befehl** – Befehle auf VMs/LXC
- **Power-Aktion** – VM/LXC-Energieverwaltung nach Zeitplan

> **Erfordert** Plus-Lizenz.
