# Pool anlegen / bearbeiten

Pools setzen Ressourcen-Quoten für eine Gruppe von Nutzern.

## Felder
- **Name** – kurzes Label
- **Quoten** – max. VMs, vCPUs, RAM (GB), Disk (GB). Leer = unbegrenzt.
- **Mitglieder** – Nutzer und Gruppen zum Pool hinzufügen

> **Erfordert** Plus-Lizenz.
