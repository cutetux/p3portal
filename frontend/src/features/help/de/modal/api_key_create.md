# API-Key erstellen

Persönliche API-Keys (Präfix `upk_`) ermöglichen externen Tools den Zugriff auf die Portal-API.

## Felder
- **Name** – beschreibendes Label
- **Scopes** – welche API-Bereiche der Key nutzen darf
- **Ablauf** – optionales Ablaufdatum

## Nach der Erstellung
Der vollständige Key wird **einmalig** angezeigt. Sofort kopieren – er kann später nicht mehr abgerufen werden.

> Core-Edition: max. 1 API-Key pro Nutzer. Plus: unbegrenzt.
