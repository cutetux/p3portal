# Packer-Build-Formular

Starte einen neuen Packer-Build, um ein VM-Template auf Proxmox zu erstellen.

## Felder
- **Build-Definition** – wähle eine Packer-Konfiguration
- **Ziel-Node** – Proxmox-Node, auf dem das Template erstellt wird
- **Parameter** – Version, Disk-Größe, Pakete usw.

## Build-Dauer
Der Build dauert typischerweise 5–30 Minuten je nach OS und Paketliste.
