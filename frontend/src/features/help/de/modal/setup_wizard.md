# Setup-Assistent

Der Setup-Assistent führt dich durch die Erstkonfiguration des Portals.

## Schritte
1. **Lizenz** – Edition einsehen, Plus-Lizenz hochladen
2. **Datenbank** – SQLite oder PostgreSQL wählen
3. **Admin-Konto** – Benutzername und Passwort setzen
4. **Proxmox-Node** – ersten Node oder Cluster hinzufügen
5. **API-Tokens** – Tokens für den Node eingeben
6. **Packer** – optionaler Packer-API-Token
7. **Fertig** – Portal startet und loggt dich automatisch ein

## Wizard erneut ausführen
Der Wizard wird nur einmalig angezeigt. Einstellungen danach über **System Settings** ändern.
