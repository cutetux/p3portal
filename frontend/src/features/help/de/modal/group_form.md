# Gruppe anlegen / bearbeiten

Gruppen bündeln Nutzer für einfachere Rechteverwaltung.

## Felder
- **Name** – eindeutiger, beschreibender Name
- **Mitglieder** – lokale Portal-Nutzer hinzufügen
- **Tags** – optionale Etiketten zur Filterung

## Geerbte Rechte
Der Gruppe kann ein Rollenpreset und Node-Zugriffsrechte zugewiesen werden. Alle Mitglieder erben diese Rechte.

> **Core-Edition:** max. 3 Gruppen.
