# Alert-Preset-Formular (Plus)

Alert-Presets sind wiederverwendbare Sets von Alarmregeln für mehrere VMs.

## Felder
- **Preset-Name** – Anzeigename
- **Regeln** – jede Regel: Metrik, Schwellwert, Dauer, Schweregrad

## Preset zuweisen
Gehe zur Detailseite einer VM → Alerting-Tab → Preset zuweisen.
