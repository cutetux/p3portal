# ISO herunterladen / verwalten

ISO-Images direkt vom Portal aus auf Proxmox-Nodes verwalten.

## ISO herunterladen
1. Ziel-Node wählen
2. Storage auswählen
3. Download-URL eingeben
4. **Herunterladen** klicken

## ISO löschen
Klicke auf das Lösch-Icon neben einem ISO. Die Datei wird vom Proxmox-Datenspeicher entfernt.
