# Zeitgesteuerter Job

Erstelle einen neuen zeitbasierten Job.

## Job-Typ
- **Ansible-Playbook** – Playbook nach Cron-Zeitplan
- **SSH-Befehl** – Shell-Befehl per SSH auf einer VM/LXC
- **Power-Aktion** – VM/LXC starten/stoppen/neu starten

## Zeitplan
Visueller Cron-Picker oder direkter Cron-Ausdruck. Zeiten in der lokalen Serverzeit.
