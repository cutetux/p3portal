# Image Factory – LXC Templates

Lade LXC-Container-Templates (`.tar.gz` / `.tar.zst`) über alle konfigurierten Proxmox-Nodes hinweg herunter, hoch und verwalte sie.

## Aus Online-Repositories herunterladen
Gib eine URL ein (z.B. vom Proxmox Download-Helper oder pveam), um ein Template direkt auf den Speicher eines Nodes herunterzuladen.

## Lokale Datei hochladen
Lade eine Template-Datei von deinem Computer auf den ausgewählten Node und Speicher hoch.

## Template löschen
Wähle ein Template aus der Liste und lösche es. Nur Administratoren können Templates löschen.

## Speicherauswahl
Templates werden pro Node gespeichert. Verwende die Node- und Speicher-Dropdowns, um Templates auf dem richtigen Ziel zu verwalten.
