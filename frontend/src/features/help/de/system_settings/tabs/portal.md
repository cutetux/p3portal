# Portal-Einstellungen

Allgemeine Konfiguration des Portals selbst.

## Sektionen
- **Erscheinungsbild** – aktives Theme auswählen, Logo hochladen
- **Sprachen** – verfügbare UI-Sprachen verwalten (Plus: eigene Übersetzungen hochladen)
- **API-Keys** – externen API-Zugang konfigurieren
- **Proxmox-Login** – nativen Proxmox-Login aktivieren/deaktivieren
- **Freigabe-Workflow** – Master-Toggle für das 4-Augen-Prinzip
- **SSH-Keys** – Standard-SSH-Keys für VM-Deployments
