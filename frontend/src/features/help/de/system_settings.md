# System Settings

System Settings ist der zentrale Konfigurationsbereich für Portal-Administratoren.

## Tabs
- **Portal** – allgemeine Einstellungen, Erscheinungsbild, API-Keys, Monitoring
- **Nodes** – Proxmox-Nodes verwalten
- **Nutzer & Rechte** – Nutzer, Rollenpresets, Pools, Gruppen, Playbook-Rechte
- **Inhalte** – hochgeladene Playbooks, Packer-Definitionen und Hilfetexte
- **Monitoring** – globale Alarmregeln (Plus)
- **Freigabe-Workflow** – 4-Augen-Prinzip konfigurieren

Der Zugriff auf jeden Tab wird durch Portal-Berechtigungen gesteuert.
