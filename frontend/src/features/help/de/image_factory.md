# Image Factory

Die Image Factory verwaltet LXC-Container-Templates (`.tar.zst` / `.tar.gz`) auf deinen Proxmox-Nodes.

## Templates anzeigen
Die Seite zeigt alle LXC-Templates auf jedem konfigurierten Node.

## Herunterladen
URL eingeben und auf **Herunterladen** klicken. Der Download läuft im Hintergrund auf dem Proxmox-Node.

## Hochladen
Lokal erstellte Template-Dateien direkt auf einen Node hochladen.
