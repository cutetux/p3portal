# System Settings

System Settings is the central configuration area for portal administrators.

## Tabs
- **Portal** – general settings, appearance, API keys, monitoring alerts
- **Nodes** – add and manage Proxmox nodes / clusters
- **Users & Rights** – manage users, role presets, pools, groups and playbook permissions
- **Content** – manage uploaded playbooks, Packer definitions and help text overrides
- **Monitoring** – global alert rules (Plus)
- **Approval Workflow** – configure the 4-eyes approval process

Access to each tab is controlled by portal permissions (manage_settings, manage_nodes, manage_users).
