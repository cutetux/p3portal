# Setup Wizard

The Setup Wizard guides you through the initial configuration of the portal.

## Steps
1. **Licence** – review edition information and optionally upload a Plus licence
2. **Database** – choose SQLite (default) or PostgreSQL
3. **Admin account** – set the initial admin username and password
4. **Proxmox node** – add your first Proxmox node or cluster
5. **API tokens** – enter the API tokens for the node
6. **Packer** – optional: enter a Packer API token
7. **Done** – the portal starts and logs you in automatically

## Re-running the wizard
The wizard is only shown once. To change settings afterwards, use **System Settings**.
