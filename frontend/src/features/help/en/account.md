# My Account

Your personal account settings.

## Tabs
- **Profile** – display name, email
- **Appearance** – personal theme and language preference
- **Notifications** – email notification settings
- **Security** – change password, view active sessions
- **SSH Key** – your default SSH public key (used in VM deployments)
- **Sessions** – active login sessions across devices
- **API Keys** – personal API tokens for external automation
- **Groups** – the groups you are a member of
- **Pools** – the resource pools you belong to
- **My Resources** – VMs and containers you own
- **Favourites** – pinned sidebar shortcuts
- **My Requests** – approval requests you have submitted
- **My Help Texts** – your personal Markdown overrides for the help system
