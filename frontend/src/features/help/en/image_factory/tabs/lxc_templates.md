# Image Factory – LXC Templates

Download, upload, and manage LXC container templates (`.tar.gz` / `.tar.zst`) across all configured Proxmox nodes.

## Download from online repositories
Enter a URL (e.g. from the Proxmox download helper or pveam) to download a template directly to a node's storage.

## Upload a local file
Upload a template file from your computer to the selected node and storage.

## Delete a template
Select a template in the list and delete it. Only administrators can delete templates.

## Storage selection
Templates are stored per node. Use the node and storage dropdowns to manage templates on the correct target.
