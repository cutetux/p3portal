# Logs & Events

The Logs & Events page shows the history of all jobs run through the portal.

## Tabs
- **Events** – Ansible and Packer job history with status, user and duration
- **Proxmox Audit** – raw API calls made to Proxmox (opt-in, enable in Portal settings)
- **Portal Audit** – security-relevant actions like logins, permission changes and API key usage

## Filtering
Use the search box and status filter to find specific jobs. Admins can filter by user.

## Live log
Click on any running or completed job to open the live log viewer.
