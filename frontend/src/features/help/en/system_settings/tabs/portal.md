# Portal Settings

General configuration for the portal itself.

## Sections
- **Appearance** – choose or customise the active theme, upload a logo
- **Languages** – manage available UI languages (Plus: upload custom translations)
- **API Keys** – configure external API access (M2M tokens)
- **Proxmox Login** – enable/disable native Proxmox user authentication
- **Approval Workflow** – master toggle for the 4-eyes approval process
- **SSH Keys** – default SSH keys injected into deployed VMs

## Themes
The portal ships with four built-in themes. Plus users can create custom themes via the Theme Editor.
