# Automation / Scheduled Jobs

The Automation page gives a global overview of all scheduled jobs across all nodes.

## Filtering
Filter by status (active, suspended, disabled) or search by name.

## Creating a job
Click **+ Job erstellen** and select the target node and job type.

## Job types
- **Ansible Playbook** – run a playbook on a cron schedule
- **SSH Command** – execute commands on VMs/LXC
- **Power Action** – VM/LXC power management on a schedule

## Approval
Jobs requiring approval appear with a *pending* badge. An approver must confirm before the job runs.

> **Requires** Plus licence.
