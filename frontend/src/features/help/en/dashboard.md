# Dashboard

The dashboard gives you a real-time overview of your Proxmox cluster.

## What you see
- **Node cards** – CPU, RAM and storage utilisation per compute node
- **VM table** – all virtual machines and containers you have access to with status, IP address and resource usage
- **Announcements** – admin-pinned messages at the top of the page

## Tips
- Click a node card to filter the VM table to that node
- Click a VM name to open the detail page
- The cluster status bar at the top shows a summary of the entire cluster
