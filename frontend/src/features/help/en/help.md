# P3 Handbuch

Welcome to the P3 Portal manual.

## Using the manual
- Click the **(i)** icon on any page, tab or modal to open the contextual help
- Use the **back arrow** in the help panel to navigate between linked topics
- Click a category below to browse topics

## Customising help texts
You can upload your own Markdown file to replace any built-in help text:
1. Open the relevant help topic
2. Scroll to the footer of the help panel
3. Select language, upload your `.md` file and confirm the rights transfer

Your personal override is only visible to you. Admins with `manage_help` can promote an override to make it visible to all users.

## Search
Use the search box at the top of this page to find topics by title or content.
