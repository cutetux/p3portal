# Pools (Resource Quotas)

Pools are portal-internal resource containers that limit how many VMs, CPUs, RAM and disk a group of users can deploy.

## Creating a pool
Go to **System Settings → Users & Rights → Pools** and click **+ Pool erstellen**.

## Quotas
Set maximum values for: VMs, vCPUs, RAM (GB) and disk (GB). Leave blank for unlimited.

## Assigning users
Add users or groups to a pool. When they deploy, the pool quota is checked before the job starts.

> **Note:** Pools require a Plus licence. Core edition: 0 pools.
