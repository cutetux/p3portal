# Compute Nodes

The Compute Nodes page shows detailed information about each registered Proxmox node.

## Node cards
Each card shows real-time CPU, RAM and storage usage. Click a card to expand the detail view.

## Tabs per node
- **Details** – hardware info, uptime, kernel version
- **VMs & LXC** – all guests on this node with status and resource usage
- **Templates** – VM templates stored on this node
- **Events** – Proxmox task log for this node
- **Backups** – backup jobs and their results
- **Alerting** – configure threshold alerts (Plus)
- **Scheduled Jobs** – time-triggered actions (Plus)
